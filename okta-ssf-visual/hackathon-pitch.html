Technical Depth

What makes this hard

Standards

Full SSF / RFC 8936 Implementation

Standards-compliant Security Event Tokens with secevent+jwt type header, proper audience validation, and Okta-specific schema compliance.

Cryptography

Browser-Side RSA Key Generation

RS256 keypair generation in the browser via Web Crypto API. Private key exported as PKCS8 for server-side JWT signing. JWKS export for Okta verification.

Architecture

Production-Grade Next.js App

React 19 with experimental Compiler, TypeScript strict mode, config-driven provider system, session persistence, and real-time activity logging.

Automation

Scenario & Bulk Orchestration

Pre-built multi-step attack scenarios with configurable delays. Bulk event queue with progress tracking. Transmission replay from history.

AI-Assisted Development

Built with Claude Code

90%+ of this codebase was generated through iterative AI-assisted development with Claude Code — from the RSA crypto pipeline to the complete component architecture. The AI understood the SSF specification, Okta's API requirements, and React 19 patterns to produce production-quality code.
